# ☕ Coffee Shop Ordering System (Python)

This is a simple beginner-level Python project that simulates a coffee shop ordering system.  
The user can choose items from the menu, specify quantity, and get a final bill.

---

## 📌 Features

- Display full menu (categories + items + prices)
- Take user orders
- Accept multiple orders in one session
- Calculate total price automatically
- Show error message if item is not found
- Generate final bill

---

## 🧠 Concepts Used

- Python dictionaries
- Loops (`while`, `for`)
- Conditional statements (`if`)
- User input handling
- String methods (`strip()`, `lower()`)

---

## ▶️ How to Run

1. Make sure Python is installed on your system  
2. Download or clone the repository  
3. Run the program:

```bash
python coffee_shop.py
```
